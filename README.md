# Text-Based-DND-Game
Text-Based Dungeons and Dragons Game[^1]

  - the game is played in IDE console
  
  - there are 4 music files[^2]
    1. for general ambiance
    2. for enemy combat
    3. for mini boss combat
    4. for boss combat
    
  - story/dialogues are also written by our group[^3]
